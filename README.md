# 即刻目录

即刻目录是一个基于 FastAPI 的 PDF 目录处理工具，支持以下流程：

1. 上传 PDF 并预览指定页面。
2. 调用兼容 OpenAI 的视觉模型识别目录文本。
3. 人工编辑目录内容后，一键生成带书签目录的新 PDF。
4. 按页码区间拆分 PDF。

前端为纯静态页面（`static/`），后端提供 API（`main.py`），可本地运行或通过 Docker 部署到 Zeabur 等平台。

## 功能概览

1. 目录识别（OCR + 结构化输出）
2. 目录写入 PDF（生成书签 TOC）
3. 局部偏移规则（支持 `--- OFFSET: 数值 ---`）
4. PDF 区间拆分
5. 兼容 OpenAI 风格接口（可配置 `API_BASE_URL`、`API_KEY`、`MODEL_NAME`）

## 环境要求

1. Python 3.10+
2. 可访问目标大模型接口（用于目录识别）

## 本地启动

1. 安装依赖：

```bash
pip install -r requirements.txt
```

2. 启动服务：

```bash
python main.py
```

3. 浏览器访问：

```text
http://127.0.0.1:8080
```

## Docker 启动

1. 构建镜像：

```bash
docker build -t jike-toc .
```

2. 运行容器：

```bash
docker run -p 8080:8080 jike-toc
```

## 使用方法

### 1. 配置模型参数

在页面顶部配置区域填写：

1. `API_BASE_URL`：兼容 OpenAI 的接口地址（如 `https://api.openai.com/v1`）
2. `API_KEY`
3. `MODEL_NAME`：需支持图片输入

点击“保存配置”。

### 2. 上传 PDF

将 PDF 拖入上传区，系统会读取页数并启用后续按钮。

### 3. 设置预览范围

填写：

1. 开始提取页
2. 结束提取页

点击“加载页面”，预览区会渲染对应页。

### 4. 识别目录

点击“AI 识别目录”，模型会基于预览页输出目录文本到编辑器。

### 5. 编辑目录并设置目录起始页

目录文本格式示例：

```text
第一章 绪论 1
    1.1 研究背景 3
    1.2 研究意义 5
```

支持局部偏移：

```text
--- OFFSET: 2 ---
第二章 方法 10
```

`目录起始页（目录第1页对应PDF第几页）` 说明：

1. 如果目录中的“1”对应 PDF 第 9 页，则填写 `9`。
2. 系统计算逻辑为：`真实页 = (目录页码 - 1) + 目录起始页 + 局部OFFSET`。

### 6. 生成最终 PDF

点击“生成最终 PDF”，浏览器将下载带目录书签的新文件。

### 7. 拆分 PDF（可选）

在“附加工具”中填写起止页码，点击“提取页面”即可下载拆分结果。

## API 说明

1. `POST /api/recognize`：识别目录文本
2. `POST /api/generate_pdf`：写入目录并返回 PDF
3. `POST /api/split_pdf`：按区间拆分 PDF

## 常见问题

### 1. 生成 PDF 失败

建议依次检查：

1. 目录文本格式是否正确（每行末尾建议带页码）
2. `目录起始页` 是否填写合理
3. PDF 是否加密或损坏
4. 浏览器控制台和服务日志中的具体错误信息

### 2. 下载器（如 IDM）弹出但无进度

这通常与下载器对 `POST + fetch` 请求的接管有关。可临时关闭该站点接管后重试。

### 3. 中文文件名下载报编码错误

项目已对 `Content-Disposition` 做 UTF-8 兼容处理。如仍出现，请确认线上已部署最新版本。
